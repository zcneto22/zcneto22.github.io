# **为什么会有这个网站**
---

> [!TIP]
>  本站取名 **r2coding**，即 **Road To Coding**，意为「编程自学之路」，是自学编程以来所用资源和分享内容的大聚合。
>  
>  旨在为编程自学者提供一系列：
>  - **清晰的编程自学路线**
>  - **各编程岗位方向的知识点梳理**
>  - **靠谱的资源**
>  - **高效的工具**
>  - **务实的技术文章**t 
>
> 方便自己也方便他人。网站内容会保持**持续更新**，欢迎收藏品鉴！
